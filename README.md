# TRM | ACL Testing & Outcome Measures

A clinical tool for ACL rehabilitation testing, documentation, and physician communication built for Train Recover Move.

## Features

- **Full testing battery** — ROM, quad girth, knee extension strength, isometric torque (HHD), Y-Balance, Vald ForceDecks data entry, 4-hop test battery, pro agility
- **Auto-calculated metrics** — LSI, Quadriceps Index, normalized torque, Y-Balance composite scores, anterior reach asymmetry flag
- **SOAP note generator** — one-click objective section, copy straight to EMR
- **Progression criteria** — real-time phase readiness evaluation (Return to Running through Phase 4 Plyometrics)
- **Session comparison** — paste previous objective data, get side-by-side table + auto-written progress paragraph
- **Physician letter** — fully auto-populated from testing data, only clinical impression requires manual input

## No dependencies on external services

- No AI API required
- No login or subscription
- Works in any modern browser
- Single-page app — runs entirely client-side

## Tech Stack

- React 18
- Vite 5
- Pure CSS-in-JS (no external UI libraries)
